## Project Overview

This is a simple, command-line calculator application written in C#. It's a console application that prompts the user for two numbers and an arithmetic operator (+, -, *, /), performs the calculation, and displays the result. The application runs in a loop, allowing for multiple calculations until the user decides to exit.

This project is built using .NET 10.0.

## Building and Running

To build and run this project, you need the .NET SDK installed.

1.  **Build the project:**
    ```bash
    dotnet build
    ```

2.  **Run the project:**
    ```bash
    dotnet run
    ```
